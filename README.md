# Ukraine Book Publishing Analysis (1991–2025)

## Project Goal
Analyze trends in Ukrainian book publishing:
- total titles and print runs
- language distribution (Ukrainian vs Russian)
- thematic structure (education, fiction, non-fiction)

## Data Sources
- Book Chamber of Ukraine (ukrbook.net)
- Additional compiled datasets

## Tech Stack
- Python (pandas, numpy)
- matplotlib / seaborn
- Jupyter Notebook

## Key Questions
- How did print runs change since the 1990s?
- How did Ukrainian language share evolve?
- What is the average print run trend?
